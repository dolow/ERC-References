ERC 721 Sample

This sample provides ERC721 based token minting.

Instruction

  • Get some testnet ETH from faucet.
  • Enter any URI you want to mint with.
  • Press "Mint" button,
  • Newly minted NFT () is transfered to your wallet.
  • Import ERC721 token address to show NFT as in the wallet.

* Instruction above is based on Metamask UI.
